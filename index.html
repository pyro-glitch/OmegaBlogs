Dashboard Visualization Using Python
Introduction
Have you ever wondered what a nightmare it would be to write and check thousands of rows and columns and create a visualized data? Fear not, with the help of our application, you can do it in mere seconds.
So how does it work?
Basically you have to upload your files,which can be in any format such as pdf,word,or image. Our inbuilt OCR which works with the help of google's pytesseract engine will convert your file into a CSV format, which will be than processed, cleaned, and visualized to give you an interactive display of your data with the ability to make changes at individual datapoints.
Why is our application needed?
Our project is an automation to the daily work of analysts and data scientists who have to go through thousands of lines of data to visualize certain elements of data to uncover hidden patterns which are useful in the data mining and deep learning.Not only can our application visualize the data, but also provide an interactive page to the end users to interact with the data directly without needing to change the code to meet specific needs.
Applications
Our application has a number of industrial applications which include:
  • Data Analysis
  • Data Mining
  • Excel Sheet Automation
